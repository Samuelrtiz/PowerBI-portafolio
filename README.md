Aquí reúno los dashboards que he ido desarrollando en Power BI, enfocados principalmente en análisis comercial y operativo. Cada proyecto refleja lo que he aprendido trabajando con datos reales y ficticios: desde la limpieza y modelado hasta la construcción de métricas y el diseño del reporte final.

> ⚠️ **Nota:** La mayoría de los proyectos fueron desarrollados con datos ficticios generados con fines de práctica y aprendizaje.

---

## 🗂️ ¿Qué incluye cada proyecto?

- Archivo `.pbix` listo para abrir en Power BI Desktop
- Base de datos en Excel
- Imagen del dashboard
- README con el contexto y explicación del proyecto

---

## 🛠️ Herramientas utilizadas

- Power BI Desktop
- DAX
- Power Query
- Microsoft Excel

---

## ▶️ Cómo explorar los proyectos

1. Entra a la carpeta del proyecto que te interese
2. Lee el README para entender el contexto
3. Descarga el archivo `.pbix`
4. Ábrelo en Power BI Desktop e interactúa con el dashboard
